📌 Smart Attendance System:-
A simple web-based attendance system built using HTML, CSS, and JavaScript. It allows users to enter their name and automatically records the date and time.

🚀 Features:-
Enter name to mark attendance
Auto date & time generation
Displays records in box format
Input resets after submission

🛠️ Tech Used:-
HTML
CSS
JavaScript

📂 How It Works:-
Enter name
Click Mark Attendance
Attendance (name, date, time) is displayed
